About

TECH~MA is a web application, which presents a new automatic process for storing and maintenance of technical events data of an Educational Institute.

This project is to maintain different categories of data related to Technical Events organized and participated by the faculty of an Educational Institute. And also this project majorly used to generate timely reports based on end user request.
